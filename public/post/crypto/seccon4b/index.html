問題

$n$ , $e$ が与えられる. RSA暗号を解読せよ. ただし, $q=2p+1$ である.

$n=pq=2p^2+p+1$ なので $p$についての二次方程式を解くことで $p$, $q$ を求めることができます.

import os
from Crypto.Util.number import getPrime, isPrime

FLAG = os.getenv("FLAG", "ctf4b{*** REDACTED ***}").encode()
m = int.from_bytes(FLAG, 'big')

while True:
    p = getPrime(512)
    q = 2 * p + 1
    if isPrime(q):
        break

n = p * q
e = 65537
c = pow(m, e, n)
